# Менеджер загрузок файлов

Приложение для загрузки файлов из интернета с поддержкой многопоточности, тегов и управления загрузками.

## Возможности

- ✅ Загрузка файлов с указанием URL и пути сохранения
- ✅ Многопоточная загрузка (настраиваемое количество потоков)
- ✅ Теги для организации загрузок
- ✅ Управление загрузками: пауза, возобновление, остановка, удаление
- ✅ Поиск загрузок по тегам
- ✅ Управление файлами: переименование, перемещение, удаление
- ✅ Отображение текущих, завершённых и неудачных загрузок
- ✅ Отслеживание прогресса загрузки

## Требования

- .NET 8.0 или выше
- Windows (используется WPF)

## Сборка и запуск

```bash
dotnet build
dotnet run
```

## Использование

1. **Добавление новой загрузки:**
   - Введите URL файла
   - Укажите путь для сохранения (или используйте кнопку "Обзор...")
   - Установите количество потоков (по умолчанию 4)
   - При необходимости добавьте теги через запятую
   - Нажмите "Добавить загрузку"

2. **Управление загрузками:**
   - Выберите загрузку в списке
   - Используйте кнопки: Пауза, Возобновить, Остановить, Удалить

3. **Поиск по тегам:**
   - Введите теги в поле поиска (через запятую)
   - Результаты фильтруются автоматически

4. **Управление файлами:**
   - Для завершённых загрузок доступны: Переименовать, Переместить

## Структура проекта

- `Models/` - Модели данных
- `Services/` - Сервисы (база данных, загрузка, файлы)
- `Converters/` - Конвертеры для привязки данных
- `MainWindow.xaml` - Главное окно приложения

## База данных

Приложение использует SQLite для хранения метаданных загрузок. База данных создаётся автоматически при первом запуске.
